## How to run

### Poetry

> You need Poetry for this. [Download from here.](https://python-poetry.org/docs/#installation)

```bash
poetry install
poetry shell
python main.py
```

### Venv

```bash
python -m venv .venv
./.venv/Scripts/activate
pip install -r requirements.txt
python main.py
```
